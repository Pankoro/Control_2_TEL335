-Instrucciones:
debe ejecutar en la terminal los soguientes comandos:
npm install
npm start

Una vez hecho esto, en el navegador web se abrirá la aplicación.

-Configuraciones necesarias:
Si tiene problemas para ejecutar el comando npm install, puede
deberse a que la terminar no esta ejecutada como administrador,
es necesario esto para ejecutar estos comandos.
